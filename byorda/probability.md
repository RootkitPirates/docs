# 쓰다듬기 확률 공시
모든 확률은 다음 쓰다듬기가 진행될 때 적용됩니다. (쓰다듬기 100회째 : 성공률 100% / 쓰다듬기 101회째 : 성공률 50%)

| 누적 횟수 구간 | 성공률 | 삐짐 확률 |
| -------- | ------ | --------- |
| 0 ~ 100 | `100%` | - |
| 100 ~ 300 | `(100 / (1 + ceil(x / 100)))%` 또는 `1%` 중 높은 것<br>x=누적 쓰다듬기 횟수 | - |
| 300 ~ ∞ | " | `3%` |

| 누적 횟수 | 성공률 |
| -------- | ------ |
| 100 | 50% |
| 200 | 33% |
| 300 | 25% |
| 400 | 20% |
| 500 | 16% |
| 600 | 14% |
| 700 | 12% |
| 800 | 11% |
| 900 | 10% |
| 1000 | 9% |
